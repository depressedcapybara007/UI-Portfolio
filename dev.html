Development

The goal of this assignment was to to develop an interactive interface, use interface components, and tie the components to an internal data state. Get even more comfortable with React by creating a list-based interface! Basically, an interface for users to select items out of a list, which then aggregates them.

UI/UX Designer & Web Developer.

  • Project Page: Responsive Redesign

I learned how to use JavaScript to develop a list-based interactive interface. With Material UI and React, I implemented filtering and sorting algorithms that require state variables to keep track of the changes made by the users.

Highlights

Research

Real company was chosen with extremely outdated webpage. The idea for redesign was to streamline information in a way it would be more easily understandable by users.

Design Iterations

Design went through several design stages. Low-Fi, High-Fi and Final draft was used to highlight relevant ideas.

Conclusion

This project reinforced ideas of good design and how existing product can be refactored into much more presentable state.
